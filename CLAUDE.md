# CLAUDE.md - プロジェクト概要

## プロジェクト概要
ゼネコン（鍜治田工務店、売上高800億円規模）の社長向けに、DX推進の一環として社内ポータル＋経営ダッシュボードを提案するデモサイト。
現在Lotus Notes（HCL Notes）を使用しているクライアントに対し、モダンなWebポータルへの移行イメージを見せるのが目的。

## クライアント情報
- 会社名: 鍜治田工務店
- 社是: 下げ振りの心
- 社長名: 鍜治田 八彦
- 売上高: 800億円規模の中堅ゼネコン
- 事業の特徴: 住宅（マンション）が主力だが、今後は非住宅（倉庫・学校・商業施設等）を拡大したい方針
- 現状: Lotus Notesを使用、ポータルの概念がなく画面が人によって異なる

## サイト構成
| ファイル | 内容 |
|---------|------|
| `index.html` | 社内ポータル（メインページ） |
| `dashboard.html` | 全社ダッシュボード（8セクション） |
| `slides.html` | DX推進提案スライド（reveal.js、全8ページ） |
| `dashboard-features.md` | ダッシュボード機能一覧（提案資料） |
| `css/portal.css` | ポータル用CSS |
| `css/style.css` | ダッシュボード用CSS |
| `js/portal.js` | ポータル用JS |
| `js/dashboard.js` | ダッシュボード用JS（Chart.js v4） |

## ダッシュボードのセクション構成（サイドバー順）
1. 経営サマリー（4つのKPIにドリルダウン機能あり）
2. 実績分析（エリア別・用途別・発注者別のサブタブ切替）
3. プロジェクト管理
4. 安全・品質管理
5. 人員・リソース
6. 財務・資金管理
7. 営業・受注管理
8. 環境・ESG

## 技術スタック
- HTML / CSS / JavaScript（フレームワークなし）
- Chart.js v4（CDN経由）
- reveal.js v5（スライド用、CDN経由）
- GitHub Pages でホスティング

## ブランチ構成
- `main`: 最新版（GitHub Pages公開中）
- `v1-demo`: 初版デモのスナップショット（バックアップ）

- GitHub Pages: `https://katofumie.github.io/Sample_0409/`

## デモデータの方針
- すべてサンプルデータ（架空）
- 800億円規模に合わせて全数値を整合
- 成長率・利益率などの比率は現実的な値を維持
- 高齢の社長にも見やすいよう文字サイズは大きめ（ベース16px）

## キャッシュバスター
- ブラウザキャッシュで変更が反映されない問題が頻発するため、CSS/JSファイルのパスに `?v=N` を付与している
- 変更時はバージョン番号を上げること（dashboard.html: v=3、index.html: v=2）

## 注意事項
- 実在の会社名（鍜治田工務店）がソースに含まれている（リポジトリはPublic）
- パルス点滅などリッチなアニメーションは不要（クライアント要望）
- 用途別の分析は「住宅（マンション）が主力、非住宅を今後拡大」が前提
