# ゼネコン向け 全社ダッシュボード

総合建設会社の経営層が、全社の状況をリアルタイムに把握し、迅速な意思決定を行うための統合ダッシュボードのプロトタイプです。

## デモ

`index.html` をブラウザで開くと動作します。

## 画面構成

| セクション | 概要 |
|-----------|------|
| 経営サマリー | 売上・利益KPI、受注/完工推移、部門別構成、経営アラート |
| プロジェクト管理 | 全案件ステータス、工程進捗比較、重点プロジェクト一覧 |
| 安全・品質管理 | 度数率/強度率、災害推移、品質検査レーダー、パトロール結果 |
| 人員・リソース | 技術者配置、部門別残業、資格者充足率 |
| 財務・資金管理 | キャッシュフロー、債権回収、部門別損益、コスト推移 |
| 営業・受注管理 | パイプライン、受注実績推移、注目案件 |
| 環境・ESG | CO2排出量、廃棄物内訳、ESG目標達成状況 |

## 技術スタック

- HTML / CSS / JavaScript（フレームワークなし）
- [Chart.js](https://www.chartjs.org/) v4（CDN経由）
- レスポンシブ対応（PC / タブレット / モバイル）
