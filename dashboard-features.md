# ゼネコン向け 全社ダッシュボード 機能一覧

## 概要

ゼネコン（総合建設会社）の経営層が、全社の状況をリアルタイムに把握し、迅速な意思決定を行うための統合ダッシュボード機能一覧です。

---

## 1. 経営サマリー

| # | 機能 | 説明 |
|---|------|------|
| 1-1 | 売上・利益サマリー | 全社の売上高・営業利益・経常利益を月次・四半期・年度で表示 |
| 1-2 | 受注残高推移 | 受注残高の推移グラフと前年同期比較 |
| 1-3 | 受注・完工高予実管理 | 受注高・完工高の予算vs実績をリアルタイム表示 |
| 1-4 | KPI スコアカード | 経営上の重要指標（営業利益率・自己資本比率・ROE等）を一覧表示 |
| 1-5 | 経営アラート | 予算乖離・資金繰りリスク等の異常値を自動検知し通知 |

## 2. プロジェクト管理

| # | 機能 | 説明 |
|---|------|------|
| 2-1 | 全社プロジェクト一覧 | 進行中の全プロジェクトを一覧表示（工種・規模・進捗率） |
| 2-2 | 工程進捗ヒートマップ | 各現場の工程進捗を色分けで可視化（順調/注意/遅延） |
| 2-3 | プロジェクト損益管理 | 案件別の実行予算消化率・粗利率をリアルタイム表示 |
| 2-4 | 重点プロジェクト監視 | 大型案件・リスク案件をピックアップし進捗・課題を集約表示 |
| 2-5 | 竣工予定カレンダー | 今後の竣工予定を時系列で表示し、引渡しスケジュールを管理 |

## 3. 安全・品質管理

| # | 機能 | 説明 |
|---|------|------|
| 3-1 | 安全成績ダッシュボード | 度数率・強度率・無災害日数などの安全指標を全社・現場別に表示 |
| 3-2 | 災害・事故発生状況 | 労働災害・事故の発生状況をリアルタイムでマップ表示 |
| 3-3 | 品質検査状況 | 各現場の品質検査合格率・是正対応状況を集約表示 |
| 3-4 | 安全パトロール結果 | パトロールの実施状況・指摘事項・改善率を可視化 |
| 3-5 | コンプライアンスアラート | 法令違反リスク・届出期限の超過等を自動警告 |

## 4. 人員・リソース管理

| # | 機能 | 説明 |
|---|------|------|
| 4-1 | 要員配置状況 | 技術者・作業員の現場別配置状況と過不足を可視化 |
| 4-2 | 資格保有者管理 | 監理技術者・主任技術者等の資格者配置状況と資格期限管理 |
| 4-3 | 協力会社稼働状況 | 主要協力会社の稼働状況・手配可能人員の把握 |
| 4-4 | 重機・設備稼働率 | 主要建設機械の稼働率・配置状況をリアルタイム表示 |
| 4-5 | 残業・休暇状況 | 部門別・現場別の時間外労働時間と有給取得率を表示 |

## 5. 財務・資金管理

| # | 機能 | 説明 |
|---|------|------|
| 5-1 | 資金繰り予測 | 今後3〜6ヶ月の入出金予測とキャッシュフロー推移 |
| 5-2 | 債権回収状況 | 工事代金の請求・入金状況と滞留債権のアラート |
| 5-3 | 外注費・資材費推移 | 主要コストの推移と予算対比を表示 |
| 5-4 | 部門別収支 | 建築・土木・設計等の部門別損益をリアルタイム集計 |
| 5-5 | 与信管理 | 発注者・協力会社の与信情報と取引リスクの可視化 |

## 6. 営業・受注管理

| # | 機能 | 説明 |
|---|------|------|
| 6-1 | 案件パイプライン | 営業案件の進捗（情報収集→提案→見積→受注）をファネル表示 |
| 6-2 | 入札状況管理 | 参加中の入札案件一覧と結果の集計 |
| 6-3 | 受注確度分析 | 案件ごとの受注確度をスコアリングし一覧表示 |
| 6-4 | 顧客別受注実績 | 主要顧客別の受注実績推移とリピート率 |
| 6-5 | 市場動向レポート | 建設投資動向・公共事業予算などの外部指標を表示 |

## 7. 環境・サステナビリティ

| # | 機能 | 説明 |
|---|------|------|
| 7-1 | CO2排出量モニタリング | 全社・現場別のCO2排出量と削減目標の達成状況 |
| 7-2 | 廃棄物管理 | 建設廃棄物の排出量・リサイクル率を集計表示 |
| 7-3 | グリーン調達率 | 環境配慮型資材の調達比率を追跡 |
| 7-4 | ESG指標サマリー | ESG関連KPIの達成状況を一覧表示 |

## 8. 共通機能

| # | 機能 | 説明 |
|---|------|------|
| 8-1 | ロールベースアクセス制御 | 役職・権限に応じた閲覧範囲の制御 |
| 8-2 | ドリルダウン | サマリーから現場単位・明細単位へ掘り下げ表示 |
| 8-3 | カスタムレポート出力 | 任意の期間・条件でPDF/Excel形式のレポートを生成 |
| 8-4 | モバイル対応 | タブレット・スマートフォンでの閲覧に最適化 |
| 8-5 | アラート・通知設定 | 閾値超過や重要イベント発生時のメール/プッシュ通知 |
| 8-6 | 基幹システム連携 | ERP・会計・工事管理等の既存システムとのデータ連携 |
| 8-7 | データ更新頻度設定 | リアルタイム/日次/月次など画面ごとのデータ更新頻度の設定 |

## 技術要件（参考）

| 項目 | 内容 |
|------|------|
| アーキテクチャ | Webアプリケーション（SPA） |
| 認証 | SSO / 多要素認証対応 |
| データ連携 | REST API / バッチ連携 |
| 対応ブラウザ | Chrome, Edge, Safari（最新版） |
| 可用性目標 | 99.5%以上 |
| レスポンス | 画面表示3秒以内 |
